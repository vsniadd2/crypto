# Блокировка/Разблокировка пользователя - Инструкция для фронтенда

## Эндпоинты
**PUT** `/admin/users/{userId}/block` - Заблокировать пользователя
**PUT** `/admin/users/{userId}/unblock` - Разблокировать пользователя

## ВАЖНО!
В URL нужно использовать **ID пользователя** (число), а НЕ email!

## Параметры
- `userId` - ID пользователя (Long, число) в URL path

## Примеры запросов
```
PUT http://localhost:8080/admin/users/1/block
PUT http://localhost:8080/admin/users/1/unblock
```

## Headers
- `Authorization: Bearer {token}` (если требуется авторизация)

## Ответ
**200 OK** - Операция успешна
```json
{
  "username": "admin",
  "email": "admin@gmail.com",
  "roles": [...],
  "timestamp": "2025-01-01T00:00:00",
  "isActive": false
}
```

**400 Bad Request** - Пользователь уже заблокирован/разблокирован
**404 Not Found** - Пользователь не найден

## Пример (Axios)
```javascript
// Получить userId из объекта пользователя
const userId = user.id; // НЕ user.email!

await axios.put(`/admin/users/${userId}/block`);
await axios.put(`/admin/users/${userId}/unblock`);
```

## Где взять userId?
В ответе `GET /admin/users` каждому пользователю приходит ID. Используйте его!
